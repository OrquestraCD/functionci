# FunctionCI
FunctionCI is an open source CI/CD Slack Bot for AWS Lambda's.

## Installation
1. Create the `kms` stack.
2. Configure your Slack app.
3. Encrypt your Slack params.
4. Create the `app` stack.
5. Add the URLS to your Slack app.
